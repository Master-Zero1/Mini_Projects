<p align="center">
  A Python-based Antakshri game where players name countries based on the last letter of the previous country.
</p>

<hr>

<h2>Features</h2>
<ul>
  <li>Random country selection</li>
  <li>Intelligent last-letter matching logic</li>
  <li>Interactive CLI gameplay</li>
</ul>

<hr>

<h2>How It Works</h2>
<ol>
  <li>User enters a country</li>
  <li>Program extracts the last valid letter</li>
  <li>Finds a matching country starting with that letter</li>
</ol>

<hr>

<h2>Tech Stack</h2>
<ul>
  <li>Python </li>
  <li>Random Module </li>
  <li>Pycountry Module </li>
  <li>Core Data Structures (Lists, Sets) </li>
</ul>

<hr>

<h2>How to Run</h2>

<pre>

</pre>
